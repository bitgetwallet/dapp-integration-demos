# wallet adapter demo
Add multi-wallet(with BitgetWallet) and multi-chain support for dapps

## how to run 
base on [Create React App](https://github.com/facebook/create-react-app)

1. `npm install`
2. `npm start`
3. Open [http://localhost:3000](http://localhost:3000)
